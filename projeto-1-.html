Títulos

Voltar

  • h1
  • h2
  • h3
  • h4
  • h5
  • h6

As tags de títulos vão do "h1" ao "h6", com seus níveis semânticos de importância seguindo do mais "importante", respectivamente. O navegador também define o font-size padrão de acordo com a tag.

Parágrafo

Voltar
  • p

A tag p representa um bloco de parágrafo, um texto comum sem muito valor semântico.

Tags de formatação

Voltar
  • O elemento mark destaca uma parte do texto.
  • O elemento small representa um comentário/print pequeno, podendo ser usado como copyright©.
  • O elemento i representa um bloco, linha ou palavra com o atributo itálico.
  • O elemento u define um "underline" no texto.
  • O elemento strong define uma parte que será marcado como negrito.

Listas

Voltar

As listas são dividas entre duas tags, ol e ul, lista ordenada e não ordenada, respectivamente.

    Este é um exemplo de lista-ordenada (ol).
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
    Este é um exemplo de lista-não-ordenada (ul).
  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Importante notar que ambas necessitam da tag li dentro delas.

Outras tags importantes

Voltar
  • A tag a define um hyperlink, link que pode ser para outra página ou até mesmo na própria página.
  • A tag hr cria uma linha horizontal para representar uma quebra temática entre dois elementos.
  • A tag sub define um alinhamento horizontal mais baixo em relação ao padrão estabelecido pelos browsers.
  • A tag sup define um alinhamento horizontal mais alto em relação ao padrão estabelecido pelos browsers.
  • A tag blokquote indica que o texto incluído é uma longa citação

Novas tags recomendadas

Voltar
  • A tag del representa uma linha de código que foi excluída do documento, esse elemento é (não necessariamente) renderizado pelos browsers com uma linha entre o texto.
  • A tag abbr representa uma abreviação e opcionalmente fornece uma descrição completa para ela.
